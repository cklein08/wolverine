# Wolverine Mobile — EDS Prototype

Edge Delivery Services prototype site for Project Wolverine (EchoStar).

Generated by **FORGE** (Fast Orchestrated Repository & Google-drive Engine).

## Features

- 🎨 Wolverine brand theming (CSS tokens from brand brief)
- 📄 Content pages: Home, Plans, About
- 🛒 Mock commerce engine (localStorage cart, vanilla JS)
- 🛍️ Product listing (PLP) with category filters
- 📦 Product detail (PDP) with add-to-cart
- 🛒 Mini-cart slide-out drawer with reactive rendering
- ✅ Checkout with order summary
- 🎯 Lighthouse 100 target

## Quick Start

```bash
npm install
npx @adobe/aem-cli up
```

## Commerce Flow

```
PLP → PDP → Add to Cart → Mini-Cart → Checkout → Place Order
```

All commerce runs on vanilla JS with `localStorage` and Custom Events (`aem:cart-updated`).
