Wolverine Mobile — Install & use

For EchoStar testers. No technical background needed.

Open the app Install page

Before you start

  • iPhone or Android phone
  • Link or QR code from your Adobe contact
  • Internet (Wi‑Fi or cellular)
  • If the link uses a laptop address (192.168…), use the same Wi‑Fi as that laptop

Android

  1. Open the link in Chrome.
  2. Tap Open Wolverine Mobile app or Install app.
  3. Allow install if Android asks (unknown apps for Chrome).
  4. Open Wolverine Mobile from your home screen.

iPhone / iPad

  1. Open the link in Safari.
  2. Tap Share (square with arrow).
  3. Tap Add to Home Screen → Add.
  4. Open Wolverine from your home screen.

If Adobe sent a TestFlight link, install the TestFlight app from the App Store first, then use their link.

Using the app

  1. Enter email and phone (or tap Skip — demo mode).
  2. Pick a profile: Texas Family, NYC Woman, or College Student.
  3. Tap Run journey → open landing to see sample messages and the personalized website.
